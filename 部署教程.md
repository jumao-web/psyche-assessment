# 心理测评 H5 产品 · 部署教程

## 你的产品是什么？

一个纯前端的网页应用（HTML + CSS + JS），不需要服务器、不需要数据库，部署到任何静态网站托管平台就能用。

## 你需要准备什么？

- 一台电脑（你正在用的就行）
- 一个浏览器
- 大约 10 分钟

---

## 方法一：Netlify 拖拽部署（最简单，30秒搞定）⭐⭐⭐⭐⭐

**不需要注册账号，不需要装任何东西！**

### 步骤：

**1. 下载部署文件**

在当前对话中，找到 `deploy/` 文件夹，里面只有4个文件：
- `index.html`
- `app.js`
- `data.js`
- `styles.css`

把这4个文件下载到你的电脑上，放在一个文件夹里（比如桌面新建一个叫 `psyche` 的文件夹）。

> 💡 如果你找不到下载方式，也可以先下载 `psyche-assessment-deploy.zip` 压缩包，解压后得到 `deploy` 文件夹。

**2. 打开 Netlify 拖拽页面**

在浏览器中打开这个网址：

👉 **https://app.netlify.com/drop**

你会看到一个页面，中间有一块虚线框，写着 "Drag and drop your site output folder here"。

**3. 拖拽上传**

把你准备好的 `deploy` 文件夹（包含4个文件的那个）**整个拖到虚线框里**。

等待几秒钟，页面会自动上传和部署。

**4. 获取你的链接**

部署完成后，页面顶部会出现一个链接，格式类似：

```
https://random-words-12345.netlify.app
```

**这就是你的产品链接！** 点开它，你的心理测评产品就上线了。

**5. 保存链接**

⚠️ 如果你没有注册 Netlify 账号，这个链接是临时的——关闭浏览器后可能丢失。

**强烈建议**：点击页面上的 "Sign up" 注册一个免费账号，这样链接就是永久的，你还可以自定义域名。

## 方法二：Vercel 部署（免费永久，需注册）⭐⭐⭐⭐

### 步骤：

**1. 注册 Vercel 账号**

👉 打开 https://vercel.com

点击 "Sign Up"，选择 "Continue with GitHub"（用 GitHub 账号登录）。

> 如果没有 GitHub 账号，先去 https://github.com 注册一个（免费）。

**2. 创建新项目**

登录后，点击右上角 **"Add New"** → **"Project"**

**3. 上传文件**

选择 **"Import Git Repository"** 下面的选项卡里找 **"Continue with a template"**

或者更简单的方式：
- 点击页面上 **"upload"** 或 **"drag and drop"** 区域
- 把 `deploy` 文件夹拖进去

**4. 部署**

点击 **"Deploy"** 按钮，等待30秒左右。

**5. 获取链接**

部署完成后，页面会显示你的链接：

```
https://你的项目名.vercel.app
```

点开就能用了！这个链接是永久的，随时可以分享。

## 方法三：GitHub Pages（免费，需注册 GitHub）⭐⭐⭐

### 步骤：

**1. 注册 GitHub 账号**

👉 打开 https://github.com → "Sign up"

**2. 创建新仓库**

- 点击右上角 **"+"** → **"New repository"**
- 仓库名称填：`psyche-assessment`
- 选择 **Public**（公开）
- 勾选 **"Add a README file"**
- 点击 **"Create repository"**

**3. 上传文件**

- 在仓库页面，点击 **"Add file"** → **"Upload files"**
- 把 `deploy` 文件夹里的4个文件拖进去
- 点击 **"Commit changes"**

**4. 开启 GitHub Pages**

- 点击仓库页面的 **"Settings"**（设置）
- 左侧菜单找到 **"Pages"**
- Source 选择 **"Deploy from a branch"**
- Branch 选择 **"main"**，文件夹选 **"/ (root)"**
- 点击 **"Save"**

**5. 获取链接**

等待1-2分钟，页面顶部会出现：

```
https://你的用户名.github.io/psyche-assessment
```

这就是你的产品链接！

## 三种方式对比

| | Netlify 拖拽 | Vercel | GitHub Pages |
|---|---|---|---|
| **难度** | ⭐ 最简单 | ⭐⭐ 简单 | ⭐⭐⭐ 稍复杂 |
| **需要注册** | 不需要（但建议注册） | 需要 | 需要 |
| **速度** | 30秒 | 1分钟 | 2分钟 |
| **链接格式** | xxx.netlify.app | xxx.vercel.app | 用户名.github.io/xxx |
| **永久免费** | ✅ | ✅ | ✅ |
| **HTTPS** | ✅ 自动 | ✅ 自动 | ✅ 自动 |
| **自定义域名** | ✅ | ✅ | ✅ |

## 常见问题

**Q：部署后打不开？**
A：等待1-2分钟，新部署需要一点时间生效。

**Q：页面样式乱了？**
A：确保4个文件都在同一个文件夹里，文件名不要改。

**Q：链接太长不好记？**
A：注册 Netlify/Vercel 后可以自定义项目名称，链接会变短。也可以买一个域名（约¥50/年）绑定。

**Q：想修改内容怎么办？**
A：在 Netlify/Vercel 后台重新上传修改后的文件即可，会自动更新。

**Q：需要备案吗？**
A：如果你用 Netlify/Vercel/GitHub Pages 的免费域名（.netlify.app/.vercel.app/.github.io），不需要备案。如果绑定自己的域名且服务器在国内，需要备案。
