## 一、平台策略（优先级排序）

### 🥇 第一优先：小红书（核心战场）

**为什么首选小红书？**
- 用户画像匹配：25-40岁女性，关注教育/心理/成长
- 搜索意图强：用户主动搜索"学习动力""考试焦虑"
- 内容生命周期长：好笔记持续获赞1-3年
- 变现路径短：直接挂链接，转化率高

**你的优势**
- 持证咨询师身份 → 认证专业号，增加信任
- 可发布"心理咨询师说"系列内容

### 🥈 第二优先：微信视频号（私域承接）

**为什么是视频号？**
- 与公众号互通，粉丝沉淀
- 朋友圈转发，裂变效果强
- 私域流量，复购率高

### 🥉 第三优先：抖音（流量放大）

**抖音定位**
- 复用小红书内容（横版改竖版）
- 投放测试（500元预算可用于DOU+）

## 四、裂变机制设计

### 4.1 分享裂变（最简单有效）

**机制**：分享测评结果到朋友圈 → 获得8折优惠券

**落地方式**：
- 在免费结果页添加"分享解锁8折"按钮
- 用户截图分享 → 凭截图领取折扣码

**你需处理**：
- [ ] 准备好优惠券码列表（如：SAVE20-001 至 SAVE20-100）
- [ ] 用户分享后手动发码（前期人工，后期可自动化）

### 4.2 邀请裂变（社交裂变）

**机制**：邀请好友测评 → 双方各减¥2

**落地方式**：
- A用户分享专属链接给B
- B完成测评后，A获得¥2优惠券
- B首次购买也减¥2

**你需处理**：
- [ ] 记录推荐关系（微信备注：推荐人XXX）
- [ ] 定期发放优惠券

### 4.3 组团购裂变（拼团模式）

**机制**：2人成团，每人立减¥3

**落地方式**：
- 发布"拼团活动"笔记
- 用户评论区@好友组队
- 你手动处理团购订单

**你需处理**：
- [ ] 在小红书/朋友圈发布拼团活动
- [ ] 记录团购信息并发放报告

## 五、自动化流程（节省时间）

### 5.1 当前手动流程

```
用户支付 → 微信收款 → 截图发你 → 你发PDF → 完成
   ↓          ↓         ↓          ↓
  1分钟     30秒      不确定      2分钟
```

**问题**：每个订单需要5-10分钟人工处理

### 5.2 半自动化流程（推荐）

```
用户支付 → 自动回复 → 用户自助获取 → 完成
   ↓          ↓                        ↓
  1分钟     自动化                   0人工
```

**落地方式**：
- 微信公众号设置"关键词自动回复"
- 用户发送付款截图 → 自动回复"请回复邮箱地址"
- 用户发邮箱 → 你定期批量发送（每天1次）

### 5.3 完全自动化（后期）

使用小程序或第三方工具实现：
- 支付后自动发送下载链接
- 用户自助下载PDF
- 零人工干预

**推荐工具**：
- 微店/有赞（简单）
- 自己开发小程序（需要技术）
