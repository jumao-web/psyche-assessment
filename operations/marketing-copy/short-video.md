# 短视频脚本合集

## 一、中高考学习动力评估

### 脚本1：痛点开场型（30秒）

**场景**：学生坐在书桌前发呆，手放在书上却没翻开

| 画面 | 文案/旁白 |
|------|----------|
| 学生坐在书桌前，盯着书发呆 | 坐在书桌前两小时，书一页没翻？ |
| 镜头拉远，显示手机在旁边 | 不是你不想学，是动力系统需要检修了 |
| 切换到测评界面 | 5分钟，测测你是哪种学习动力类型 |
| 显示四种类型卡片 | 内驱型、目标导向型、混合型、还是迷茫型？ |
| 显示测评二维码 | 找到属于你的学习节奏，点击下方链接测试 |

---

### 脚本2：对比反差型（30秒）

**场景**：两个人物对比

| 画面 | 文案/旁白 |
|------|----------|
| 画面左边：学生A开心看书 | 有人学习是这样的—— |
| 画面右边：学生B痛苦看书 | 有人学习是这样的…… |
| 同一个人，不同状态 | 同一个人，为什么有时候想学，有时候不想？ |
| 显示大脑示意图 | 答案在你的动力来源 |
| 显示测评界面 | 内在驱动 vs 外在推动，测测你的动力类型 |
| 结尾定帧 | 点击链接，找到你的学习开关 |

---

### 脚本3：数据开场型（20秒）

**场景**：数据可视化动画

| 画面 | 文案/旁白 |
|------|----------|
| 柱状图动画 | 研究发现：内驱型学生成绩更稳定 |
| 数字跳动 | 持续性比目标型高30% |

| 结尾 | 点击下方链接 |

## 拍摄建议

1. **人物选择**：
   - 学习动力/考试焦虑：中学生形象，校服或休闲装
   - 家庭教育：家长+孩子组合，真实自然
   - 青少年成长：阳光向上的青少年形象

2. **场景设置**：
   - 书房、教室、客厅等日常场景
   - 避免过度布置，保持真实感

3. **音乐/音效**：
   - 学习动力：轻松活泼的BGM
   - 考试焦虑：紧张→舒缓的情绪变化
   - 家庭教育：温馨感人的氛围
   - 青少年成长：积极向上的节奏

4. **字幕设计**：
   - 重点信息用醒目颜色
   - 关键词放大强调
   - 结尾清晰展示测评链接/二维码
